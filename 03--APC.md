# Atari Punk Console

Atari Punk Console (アタリ・パンク・コンソール)、略してAPCは、レトロゲームのような音を出力するシンセサイザー回路。

## パーツの繋げ方

今回用意したAPCボードは以下のパーツを接続することで、機能する<br>
- **Battery** : 電池ボックス(単三４本、6V) x 1個
- **Trimmer01 & Trimmer02** : トリマー(可変抵抗) x 2個
- **Switch** : スイッチ x 1個
- **Speaker** : 小型スピーカー x 1個
